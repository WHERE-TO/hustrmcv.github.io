# 文章内容格式为:	

### \<time\>时间\</time\>

### \<tit\>标题\</tit\>

### \<article\>  

	文章内容  

	<p>段落</p>  

	<a href = "（文档、博客等）链接地址">一段描述</a>  

	<img src = "（图片地址）" />  

	<pre>代码</pre>  

	<li>点标号（分类、层）</li>  

### \</article\>
### \<tag\>\<part\>标签\</part\>\<part\>标签\</part\>\</tag\>
### \<more\>文章相对根目录地址\</more\>
